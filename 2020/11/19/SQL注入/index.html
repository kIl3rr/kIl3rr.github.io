  • xpath

    XPath 即为 XML 路径语言，是 W3C XSLT 标准的主要元素，它是一种用来确定 XML（标准通用标记语言的子集）文档中某部分位置的语言。

    XPath 基于 XML 的树状结构，有不同类型的节点，包括元素节点，属性节点和文本节点，提供在数据结构树中找寻节点的能力，可用来在 XML 文档中对元素和属性进行遍历。

    XPath 使用路径表达式来选取 XML 文档中的节点或者节点集。这些路径表达式和我们在常规的电脑文件系统中看到的表达式非常相似。

    XPath是一种用来在内存中导航整个XML树的语言,它的设计初衷是作为一种面向XSLT和XPointer的语言,后来独立成了一种W3C标准.

    详细的可以去学习xml 所谓xpath攻击就是利用xpath解释器的松散输入和容错特性，能够在url、表单等插入恶意的xpath查询代码 本质上与sql注入没什么区别 只是注入的目标不再是数据库 而是xml文档 值得注意的是xml没有访问控制和用户认证，我们不会碰到在sql注入中遇到的许多访问限制的问题 ，所以能够利用xpath查询访问整个xml文档 注入口在cookie，header，request paraments/input

    因为这题并不是完全没有回显 我们可以试一试updatexml报错注入

updatexml(xml_document,xpath_string,new_value)

第一个参数是string格式，xml对象的名称；第二个参数是xpath的字符串；第三个参数是string格式替换查找到的符合条件的数据
